Combolist(optList, defaultValue)

Creates a combolist, containint values from optList array, with a value of defaultValue. Its main div is accessible as Combolist::div. Onchange callback may be set via Combolist::onchange property.

Combolist::addOption(text)

Adds a new option with value of text.

Combolist::clearOpts()

Clears all options.

var cl = new Combolist([],"default");
document.body.appendChild( cl.div );
cl.addOption("option 1");
cl.addOption("option 2");
