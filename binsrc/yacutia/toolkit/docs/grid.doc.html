Grid(element, autoNumber)

Creates a grid as a child of element. If autoNumber is true, than the Grid will automatically add first, auto-numbered column.

Grid::createRow(data, [index])

Adds a row. If index is present, it specifies at which index new row should be inserted. Data is an array of either values or objects. If objects are used, they should contain:

  • value - cell value
  • align - one of ALIGN_ constants

Grid::createHeader(data)

Creates a header row. Data is an array of either values or objects. If objects are used, they should contain:

  • value - cell value
  • sortable - whether this column is sortable
  • draggable - whether this column is draggable
  • resizable - whether this column is resizable
  • align - one of ALIGN_ constants
  • sort - one of SORT_ constants
  • type - one of TYPE_ constants

Grid::appendHeader(data, [index])

Adds a new header cell. If index is present, it specifies at which index new cell should be inserted. Data are in same format as in createHeader.

Grid::fromTable(table)

Converts existing table into grid.

Grid::row::addCell(data, [index])

Adds one cell to one row. If index is present, it specifies at which index new cell should be inserted. Data are in same format as in createRow. This method should be called as grid.rows[i].addCell().

Grid::removeColumn(index)

Removes one column from grid.
